Predefined and configurable CSS scroll animations that drop straight into your projects HTML.

How To Install?

All you need to do is add the dist trig.js and trig-animations.css file into your projects JS and CSS folders then add the following code with your trig.js and trig-animations.css file paths in. Put this code in to your head HTML tags. Download Trig.js here or use npm.

                            
                                <link rel="stylesheet" href="/css/trig-animations.css"></link>
                                <script src="/js/trig.js"></script>

Or just add a CDN instead

                            
                            <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/trig-js/src/trig-animations.min.css"></link>
                            <script src="https://cdn.jsdelivr.net/npm/trig-js/src/trig.min.js"></script>
